# 完善python驾驶舱核心功能
1. 代码转换功能，目前的前端页面已经开发完毕，需要你完善后端代码，实现代码转换功能。使其具备任何java代码转换到python代码的能力。并且支持三种形式，分别是直接代码转换，单文件转换，多文件转换即目录。
2. 语法学习功能，需要你完善前端页面，实现python语法学习功能。用户可以在前端页面查看python的语法规则，并且可以通过代码示例来学习。需要包含所有核心语法。以及对应的代码示例。
3. 知识库功能，这是应用python的各种场景下的使用案例，需要包含各种业务场景，需要全面且丰富。每个场景下需要包含python代码示例，以及对应的业务场景。
4. 语法学习有对应的学习进度记录，用户可以在学习过程中记录自己的学习进度，并且可以在任何时候查看自己的学习进度。
5. 学习统计功能要使用真实的数据，不能使用假数据。
6. 如果发现服务端口冲突，可以进行漂移，但最多漂移三次。
