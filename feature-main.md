# python语言学习驾驶舱系统

## 项目概述
开发一个基于SpringBoot+Vue3的Python语言学习驾驶舱系统，专为Java开发者转型Python设计。系统采用高科技风格UI，提供三大核心功能模块：Java-to-Python代码转换器、Python核心语法学习中心（以Java视角）、Python全场景应用知识库。目标是帮助Java开发者快速掌握Python语言生态与应用。

## 技术栈要求
**后端**:
- Spring Boot 3.x (Java 17+)
- Maven项目管理
- Spring WebFlux（响应式编程）
- Spring Data JPA
- H2/MySQL数据库
- JWT身份认证
- Jython或自定义AST解析器用于代码转换

**前端**:
- Vue 3 (Composition API)
- TypeScript
- Vite构建工具
- Pinia状态管理
- Vue Router 4
- Element Plus或Ant Design Vue组件库
- Monaco Editor（代码编辑器）
- ECharts或D3.js（数据可视化）
- Tailwind CSS或自定义CSS实现高科技UI

**部署**:
- Docker容器化部署
- Nginx反向代理配置
- 前后端分离架构

## 功能模块详细需求

### 1. Java-to-Python代码转换工作站
- **左侧输入区**:
  - 支持单文件上传（.java）
  - 支持目录结构上传（ZIP格式）
  - 实时代码编辑器（Monaco Editor）支持Java语法高亮
  - 代码片段历史记录功能
- **右侧输出区**:
  - 实时显示转换后的Python代码（语法高亮）
  - 转换质量评分与潜在问题提示
  - 代码下载功能（单文件/目录结构）
  - 转换参数配置面板（缩进大小、命名风格转换等）
- **核心转换能力**:
  - 基础语法映射（类、方法、控制流）
  - Java OOP到Python OOP的转换
  - 异常处理机制转换
  - 集合类型智能映射（List, Map等）
  - 处理Java特有概念（如checked exceptions）到Python惯例的转换
  - 生成必要的注释说明关键转换点

### 2. Python核心语法学习中心（Java开发者视角）
- **学习路径导航**:
  - 结构化学习路径（基础→进阶→专家）
  - 每个知识点标注对应的Java概念
- **对比学习卡片**:
  - 左侧显示Java实现，右侧显示Python实现
  - 重点差异高亮标注
  - 交互式代码示例，可直接修改并运行
- **核心概念覆盖**:
  - 基础语法对比（变量、类型、控制流）
  - OOP范式对比（类、继承、多态）
  - 函数式编程（lambda、高阶函数）
  - 异常处理机制
  - 模块与包管理系统
  - 特色功能（装饰器、生成器、上下文管理器）
  - 性能优化技巧
- **交互式学习**:
  - 每个概念配小型练习
  - 智能提示与错误诊断
  - 学习进度跟踪

### 3. Python全场景应用知识库
- **场景分类导航**:
  - Web开发（Django/Flask/FastAPI对比Spring）
  - 数据科学与AI（NumPy/Pandas/PyTorch）
  - 自动化脚本与DevOps
  - 微服务与API开发
  - 桌面应用（PyQt/Tkinter）
  - 爬虫与数据采集
  - 区块链与加密货币
  - 云计算与Serverless
- **每个场景包含**:
  - 典型架构图（对比Java生态方案）
  - 核心库/框架介绍
  - 项目结构模板
  - 最佳实践指南
  - 性能考量
  - 案例研究（完整项目示例）
  - 从Java迁移的注意事项
- **全生命周期覆盖**:
  - 项目初始化与依赖管理
  - 代码规范与质量保障
  - 测试策略（单元测试、集成测试）
  - CI/CD流水线配置
  - 部署与监控
  - 性能优化与故障排查

## 特别注意事项
- 代码转换功能需要处理边缘情况并提供清晰的错误提示
- 所有Python示例代码必须遵循PEP8规范
- 学习内容必须准确对比Java与Python的思维方式差异，不仅是语法差异
- 高科技UI不应牺牲可访问性与加载性能
- 系统需要记录用户学习行为，为后续个性化推荐提供基础
- 实现完善的错误边界处理，特别是代码执行环境
